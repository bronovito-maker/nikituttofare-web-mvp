# PROGETTO: NikiTuttoFare (NTF) - Enterprise Edition
## MISSION
Trasformare NTF da chatbot a Piattaforma di Gestione Emergenze.
Focus: Fiducia, Sicurezza, Chiarezza Bancaria.

## TECH STACK (OBBLIGATORIO)
- Frontend: Next.js 14 (App Router), React Server Components.
- UI Library: Shadcn/UI (Radix Primitives) + Tailwind CSS.
- Form Management: React Hook Form + Zod (Validazione rigorosa).
- State: Zustand (per la chat), Server Actions (per il backend).
- Database: NocoDB (esistente) - Interfacciato via SDK sicuro.

## DESIGN SYSTEM "TRUST & CLARITY"
- Background: #FFFFFF (Bianco Assoluto).
- Text: Slate-900 (Primario), Slate-600 (Secondario).
- Brand: Blue-600 (Azione), Orange-600 (Urgenza/CTA).
- Radius: 0.5rem (Professional).
- NO Glassmorphism, NO Dark Mode forzata.

## ARCHITETTURA "GENERATIVE UI"
L'AI non risponde con testo libero. L'AI restituisce JSON strutturati che il frontend renderizza come Componenti React (Form, Card, Upload).

## SICUREZZA (PII PROTECTION)
- Mai salvare Nome/Telefono in chiaro nei log.
- I dati sensibili sono visibili ai tecnici solo DOPO il pagamento (Stripe).
- Magic Link per accesso tecnici (no login/password complessi).
